Chapter 6. JavaScript: beginnings

This first chapter on JavaScript covers the basics: sequences of statements, simple calculations, conditional statements and loops. You will already have covered these ideas in your Java Programming module. The aim is to make sure you understand the syntax required in JavaScript. We'll move on to more advanced ideas in later chapters.

Resources: Appendix C

We will opt for 'modern mode' JavaScript rather than 'sloppy mode' JavaScript. This means that the following will appear at the start of all JavaScript files:

"use strict";
// now the code is in strict mode

This can also be written as 'use strict';. Mozilla has more details about why we should use this.

JavaScript is covered in detail in lectures, so here we'll concentrate on some exercises. If you want a good tutorial on JavaScript to work through in advance of lectures, check out Mozilla's.

In each of the following exercises, use the prompt command to get user input:

We'll cover the use of form elements for user input in a later chapter.

The following will be useful when attempting the exercises:

  • The function parseFloat (there is also a parseInt function) can be used to convert a string into a floating point number, e.g.
    var fahrenheit = parseFloat(prompt("Fahrenheit?"));
  • The function toFixed can be used to display a floating point number using fixed point notation.
  • For exercise 1, knowledge of the division operator in JavaScript is required.
  • For exercises 4 and 5, the following character entity references will be useful: &sup2;, &sup3;, &pound;, which, respectively, display ², ³ and £.

Exercises

  1. Write a JavaScript program that reads a temperature in Fahrenheit from the keyboard and displays the equivalent in Celsius (centigrade). The formula for the conversion is: C= (F-32)*5/9. Your program should display the temperature in Celsius to an accuracy of 4 decimal places. [Solution] (Remember: clicking on the link runs the solution. When the solution has run, right click on the webpage to view the source code.)
  2. Write a JavaScript program that reads two numbers (call them x and y) input by the user and copies them into variables called largest and smallest, such that largest contains the larger of x and y and smallest contains the smaller of x and y. Your program should display a message stating which number is the larger and which is the smaller. [Solution]
  3. Write a JavaScript program that reads a number input by a user and then displays the string 'Hello' that number of times, e.g. if the user inputs 5, then 'Hello' will be displayed 5 times. [Solution]
  4. Write a JavaScript program that displays a table of the integers between 1 and 10, their squares (i.e. x²) and their cubes (i.e. x³). [Solution]
  5. Write a JavaScript program that reads three numbers: an initial sum of money, a desired sum of money and an annual rate of interest (expressed as a percentage). Your program should use a while loop to determine how many years it will take the investment to grow from the initial sum into the desired sum. [Solution]
