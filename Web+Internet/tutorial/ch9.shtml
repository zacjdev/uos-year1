Chapter 9. Graphics and interaction on the canvas

The aims of this chapter are to practise (i) drawing on the canvas; (iii) handling events related to the canvas; (iii) creating animation on the canvas.

There are four sections in this chapter:

  1. Drawing on the canvas
  2. Using buttons with a canvas
  3. Mouse clicks on the canvas
  4. Animation

9.1 Drawing on the canvas

Mozilla: canvas tutorial

(Hint: Use the programs from relevant lectures as the starting point for these exercises.)

Exercises

  1. Write a program to draw a cross in a square, as illustrated in Figure 9.1. [Solution]
  2. Find a free image (e.g. an image from the Internet). Write a program to display three subparts of the image in a row on a canvas, as illustrated in Figure 9.2. [Solution]
  3. Find five free images and draw them all on a canvas. [Solution]

9.2 Using buttons with a canvas

Exercises

  1. Create a graphical user interface that contains four buttons and a canvas. The first button should draw a red line between two random positions on the canvas when pressed. The second button should draw a green line between two random positions on the canvas when pressed. The third button should draw a blue line between two random positions on the canvas when pressed. The fourth button should clear the canvas. (Hint: use i_button1.html, available in the Appendix C download for this chapter, as a starting point.) [Solution]
  2. Input some text from the user and draw this text on the canvas. The user should also be able to input the font size of the text to display. (Hint: reuse some code from Chapter 8 to create some input areas for the user to type information into.) [Solution]

9.3 Mouse clicks on the canvas

Mozilla: Mousedown event

Exercises

  1. Draw three non-overlapping circles on the canvas. Then use the mouse to detect which circle the user clicks inside and display the result. (Hint: Use detect_in_rect1.html, available in the Appendix C download for this chapter, as a starting point.) [Solution]
  2. Draw a 2D grid pattern (as shown in Figure 9.3). Draw a circle in the grid cell that the user clicks on. (Hint: Use detect_in_grid1.html, available in the Appendix C download for this chapter, as a starting point.) [Solution]

9.4 Animation

(Hint: use the animation examples from lectures as a starting point.)

Exercises

  1. Animate the text ‘Hello’ so that it continually moves around the screen. When the user clicks on the canvas, the text should change to ‘Goodbye’. When the user clicks on the canvas again, the text should change to ‘Hello’, and so on. [Solution]
  2. Draw a circle on the canvas. When the user clicks on the canvas, the circle should move to where the user has clicked. The movement should be an animation from one point to the next rather than an instantaneous move from the previous point to the clicked point. [Solution]
